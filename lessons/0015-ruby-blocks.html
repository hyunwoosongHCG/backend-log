블록이 도대체 뭔가

지금까지 레슨에서 이런 코드를 계속 봤다: 

scope :kept, -> { where(discarded_at: nil) }

[1, 2, 3].each { |n| puts n }

rescue_from Pundit::NotAuthorizedError do |exception|
  error!('Forbidden', 403)
end

{ ... }와 do ... end — 이것들이 모두 블록(Block)이다. 블록은 "메서드에 넘겨주는 코드 덩어리"다.

JS의 콜백함수와 거의 같다:

// JS: 함수를 인자로 넘긴다
[1, 2, 3].forEach(n => console.log(n))

# Ruby: 블록을 메서드 뒤에 붙인다
[1, 2, 3].each { |n| puts n }
차이는 Ruby의 블록은 인자가 아니라 메서드 호출에 붙이는 형태라는 것. 괄호 밖에 있다.

두 가지 문법 — 같은 것이다

# 한 줄: {} 관용적으로 사용
[1, 2, 3].each { |n| puts n }

# 여러 줄: do...end 관용적으로 사용
[1, 2, 3].each do |number|
  doubled = number * 2
  puts doubled
end

|n|, |number| — 파이프(|) 사이의 것이 블록 파라미터다. 메서드가 블록을 호출할 때 넘겨주는 값을 받는 변수다.

yield — 메서드가 블록을 호출하는 방법

메서드 안에서 yield를 쓰면 "지금 붙어있는 블록을 여기서 실행해라"는 의미다. 

def greet
  puts "시작"
  yield          # 블록 실행
  puts "끝"
end

greet { puts "안녕하세요" }
# 출력:
# 시작
# 안녕하세요
# 끝

yield에 인자를 넘기면 블록 파라미터로 들어온다:

def double_it(number)
  yield(number * 2)
end

double_it(5) { |result| puts result }   # → 10

ActiveRecord의 each, map, select가 모두 이 방식으로 동작한다. 내부적으로 yield(레코드)를 호출하는 것이다.

람다(->) — 블록을 변수에 담기

블록은 보통 메서드 호출에 붙어서 쓰지만, 람다(lambda)를 쓰면 블록을 변수나 인자로 저장할 수 있다. 

# 람다 정의
double = -> (n) { n * 2 }

# 람다 호출
double.call(5)   # → 10
double.(5)       # → 10 (축약 문법)

레슨 14에서 봤던 scope 정의가 바로 이것이다:

scope :kept, -> { where(discarded_at: nil) }
#            ↑
#       인자 없는 람다 — 호출될 때 where(...)를 실행한다

인자가 없으면 -> { ... }, 인자가 있으면 -> (인자) { ... }다.

실무 코드에서 블록 읽기

이제 Grape DSL이 왜 저렇게 생겼는지 보인다:

# Grape 라우터 — 전부 블록이다
class V2::Reviews::ReviewTemplates < GrapeBase
  resource :review_templates do        # ← 블록 시작
    desc '목록 조회'
    get do                             # ← 블록 시작
      templates = ReviewTemplate.kept.order(created_at: :desc)
      present templates, with: ReviewTemplateEntity
    end                                # ← 블록 끝

    put ':id' do                       # ← 블록 시작
      authorize review_template, :update?
      review_template.update!(params)
    end                                # ← 블록 끝
  end                                  # ← 블록 끝
end

resource :review_templates do ... end — resource 메서드에 블록을 넘긴다.
get do ... end — get 메서드에 블록을 넘긴다.
Grape는 요청이 왔을 때 해당 블록을 yield로 실행한다.
