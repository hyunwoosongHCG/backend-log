# Backend Learning Resources

## Knowledge

- [Rails Guides (공식)](https://guides.rubyonrails.org/)
  Rails 공식 문서. ActiveRecord, 라우팅, 마이그레이션 등 모든 기초의 1차 출처.

- [Rails Guides 한국어](https://rubykr.github.io/rails_guides/)
  공식 가이드 한국어 번역. 초기 개념 파악에 유용하나 버전이 오래됐을 수 있음.

- [Grape 공식 README](https://github.com/ruby-grape/grape)
  Grape API 프레임워크 공식 문서. params, present, entity 등 실무 관련.

- [The Odin Project — Ruby on Rails](https://www.theodinproject.com/paths/full-stack-ruby-on-rails)
  무료 풀스택 Rails 커리큘럼. 개념 흐름 이해에 적합.

- [Ruby 공식 문서](https://ruby-doc.org/)
  Ruby 언어 자체의 클래스, 메서드, 모듈 레퍼런스.
